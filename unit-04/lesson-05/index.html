02

개념설명

⏱ 20분▾

텍스트 분류 모델

텍스트 데이터를 학습하는 과정에서 패턴을 분석하여 특정 텍스트가 어떤 범주에 해당하는지 분류합니다.

  • 스팸 메일 구분: 일반/스팸 분류
  • SNS 모니터링: 부적절 글 감지
  • 뉴스 범주: 경제·문화·연예·스포츠
  • 영화 감상평: 긍정/부정 분류

영화 평가 시스템 문제 해결 과정

문제 정의: 영화 감상평의 전반적 반응을 긍정·부정으로 자동 분류
데이터 수집: 긍정 표현("재미있다","훌륭해") + 부정 표현("별로","실망") 수집
모델 학습: 티처블 머신 텍스트 분류 모델 생성·평가
엔트리 적용: 감상평 입력 → 긍정/부정 분류 결과 출력

04

수행평가

⏱ 5분▾

📋 텍스트 분류 AI + 엔트리 영화 평가 시스템

긍정·부정 텍스트 분류 모델을 만들고 엔트리에서 영화 평가 시스템을 구현하시오.

①
긍정·부정 텍스트 데이터 각 15개 이상 수집·레이블링
②
티처블 머신으로 텍스트 분류 모델 학습·평가
③
엔트리에서 감상평 입력 → 분류 결과 출력
평가 요소 매우 잘함(4) 잘함(3) 보통(2) 노력 필요(1)
모델 성능 긍·부정 분류 정확도 75%↑ 60%↑ 모델 생성만 없음
시스템 완성도 입력→분류→출력 정상 부분 작동 연결만 없음
