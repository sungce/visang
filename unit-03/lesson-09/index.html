02

개념설명

⏱ 15분▾

함수(Function)란?

특정 기능에 이름을 붙여 묶어두고 필요할 때마다 호출하여 재사용하는 것이 함수입니다.

  • 함수 정의: 기능을 만드는 것
  • 함수 호출: 만든 기능을 사용
  • 재사용성: 같은 코드 반복 작성 불필요
  • 모듈화: 기능별로 코드 분리·관리

엔트리 함수 블록 예시

함수를 사용하는 가장 큰 이점은?
①프로그램 전체 코드 줄 수가 항상 줄어든다
②같은 코드 반복 작성 불필요, 재사용성·유지보수성 향상
③프로그램 실행 속도가 크게 빨라진다
④디버깅을 전혀 하지 않아도 된다
