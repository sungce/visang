개념설명

⏱ 25분▾

① 문자의 디지털 표현 — 문자 코드

컴퓨터는 0과 1의 이진수를 조합한 문자 코드로 문자를 표현합니다.
• 아스키(ASCII) 코드: 영문자·숫자·기호 표현 (A=1000001, B=1000010 ...)
• 유니코드(Unicode): 다양한 언어 표현 (가=AC00, 나=B098 ...)
아스키 코드 일부
A = 1000001
B = 1000010
C = 1000011
Z = 1011010
입력 → 이진수 변환 → 화면
'A' 입력
→ 1000001
→ 화면에 'A' 출력

② 이미지의 디지털 표현

🟩 비트맵(Bitmap) 방식
픽셀 단위로 이미지 표현. 색상 코드로 픽셀 정보 저장. 해상도가 높을수록 실제와 유사. 확대 시 깨짐. 파일: BMP·JPG·PNG
흑백: 0(검은색)·1(흰색)
📐 벡터(Vector) 방식
수식으로 점·선을 계산하여 이미지 표현. 확대해도 깨지지 않음. 로고·아이콘에 적합. 파일: SVG
예) 원(150,150,240) + 파란색

③ 부호화
정숫값을 이진수로 변환
💡 예) 정숫값 1→001, 2→010, 3→011, 4→100, 5→101

④ 동영상의 디지털 표현 — 프레임·FPS

• 프레임(Frame): 정지된 이미지 하나
• FPS(Frames Per Second): 1초에 보이는 프레임 수 → 동영상 품질 결정
• FPS가 높을수록 → 움직임이 자연스러움 (단, 용량 증가)
• 잔상 효과: 연속 이미지를 빠르게 넘기면 움직이는 것처럼 보임
• 파일 형식: AVI(용량 큼) / MP4(저용량 고화질) / MOV(애플)
