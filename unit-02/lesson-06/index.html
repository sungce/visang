⏱ 15분▾

급식 잔반 데이터 분석 개요

급식 잔반 측정기로 수집한 메뉴별 배식량·섭취량·잔반량 데이터를 분석하여 잔반을 줄일 수 있는 방안을 찾습니다.

  • 잔반율: (잔반량 ÷ 배식량) × 100 (%)
  • 목표: 잔반율이 높은 메뉴 파악 → 개선 방안
  • 데이터 속성: 날짜·메뉴·배식량·섭취량·잔반량·잔반율
📖 교과서 연계: pp.77~82 / 잔반율 = 잔반량 ÷ 배식량 × 100

📋 급식 잔반 감소 데이터 분석 보고서

급식 잔반 데이터를 분석하여 잔반율이 높은 메뉴와 낮은 메뉴를 비교하고, 데이터에 근거한 잔반 감소 방안 2가지 이상을 논리적으로 서술하시오.

평가 요소 매우 잘함(4) 잘함(3) 보통(2) 노력 필요(1)
데이터 분석 잔반율 계산·그래프 완성 그래프만 잔반율 계산만 없음
방안 제시 데이터 근거+2가지 이상 1가지+근거 방안만 없음
논리성 근거·설득력 명확 대체로 명확 단순 나열 없음
