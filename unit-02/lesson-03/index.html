학습목표

⏱ 5분▾
[9정02-03] 데이터 분석
문제 해결 목적에 적합한 데이터를 다양한 방법으로 수집하고, 응용 소프트웨어를 이용하여 데이터를 목적에 맞게 구분하고 관리할 수 있다. (pp.57~63)
  • ✓
    문제 해결에 적합한 데이터를 선별하고 설명할 수 있다.
  • ✓
    직접 수집(인터뷰·관찰·설문·센서)과 간접 수집(검색 엔진·웹사이트)의 차이를 설명할 수 있다.
  • ✓
    데이터 속성의 의미를 이해하고 범주형·수치형 데이터를 구분할 수 있다.
  • ✓
    코답(CODAP)을 이용하여 데이터를 관리할 수 있다.

02

개념설명

⏱ 20분▾

① 문제 해결에 적합한 데이터 선택

📱 문제: 스마트폰 사용 습관 개선
→ 사용 앱 종류·사용 횟수·사용 장소·총 사용 시간
🍽️ 문제: 급식 잔반량 줄이기
→ 식단·급식 인원·잔반량·날짜 데이터

② 데이터 수집 방법

📥 직접 수집
• 인터뷰: 질문·답변으로 의견 수집
• 관찰: 오감·관찰 도구로 기록
• 설문 조사: 설문지로 답변 모음
• 센서 이용: 빛·온도·습도 측정
장점: 목적에 맞는 정확한 데이터
단점: 시간·비용 많이 듦
🌐 간접 수집
• 검색 엔진: 인터넷 검색으로 수집
• 공공데이터포털: data.go.kr
• 국가통계포털: kosis.kr
• 캐글: kaggle.com
장점: 빠르고 편리
단점: 검증 필요·적합한 데이터 찾기 어려움

③ 데이터 속성과 구분

🔤 범주형 데이터
문자나 기호로 대상을 구분하는 데이터
예) 성별(남·여), 혈액형(A·B·O·AB), 성취 등급, 여행지 이름, 교통수단
🔢 수치형 데이터
숫자로 값을 표시하고 크기·차이를 비교하는 데이터
예) 키·몸무게, 기온·강수량, 위도·경도, 추의 무게
📖 교과서 연계: pp.57~63 / 코답(CODAP): https://codap.concord.org
