02

개념설명

⏱ 20분▾

① 데이터 구조화란?

데이터의 종류·특성·사용 목적 등에 따라 데이터를 체계적으로 구분하여 정리하는 것입니다. 구조화하면 필요한 정보를 빠르게 찾고 데이터 활용도가 높아집니다.

② 구조화 방법 3가지

📊 표(Table)
속성별로 행·열 정리. 대량 데이터 관리에 적합. 추가·삭제·수정 편리.
예) 주소록, 성적표, 여행 계획표
🌳 계층형 다이어그램
시작점에서 여러 가지로 갈라지는 구조. 분류 체계·위계 관계 파악 쉬움.
예) 학급 조직도, 폴더 구조, 공룡 분류
🕸️ 그래프형 다이어그램
노드(원·사각형)와 간선(선)으로 복잡한 관계 시각화. 직관적 파악.
예) SNS 친구 관계, 노선도, 등산로

③ 데이터 분석 — 상관관계와 산점도

상관관계: 데이터 속성 간의 관계 (한 쪽 증가 시 다른 쪽 변화)
산점도 그래프: 데이터를 좌표 평면의 점으로 표시 → 관계 한눈에 파악
• 점이 우상향으로 몰릴수록 → 양의 상관관계 (학습 시간↑ 성적↑)
• 점이 무작위로 흩어짐 → 상관관계 없음
