Method 01 · Survey

설문 조사

사람들에게 직접 질문하여 의견·경험·사실을 수집하는 방법.
종이 설문지, 온라인 폼, 인터뷰 등 다양한 형태로 진행합니다.

📋
📖 핵심 개념
1
질문 설계 — 수집 목적에 맞는 질문 문항을 만든다. 중립적이고 명확하게!
2
배포 & 응답 — 종이·구글 폼·QR코드 등으로 응답자에게 전달하여 답변을 받는다.
3
데이터 수집 — 응답이 쌓이면 숫자·텍스트 데이터가 생성된다.
4
분석 & 시각화 — 집계하여 그래프·표로 결과를 정리한다.
⚖️ 장점 & 단점
👍 PROS
짧은 시간에 많은 사람의 의견을 모을 수 있다
익명 응답으로 솔직한 답변 유도 가능
온라인 도구로 비용이 거의 없다
결과를 즉시 통계로 확인 가능
👎 CONS
응답자가 거짓 또는 부정확하게 답할 수 있다
질문 방식에 따라 결과가 달라질 수 있다 (편향)
응답률이 낮으면 데이터 신뢰도 하락
무응답·중도 포기로 결측값 발생
🌍 실생활 활용 사례
🏫
학교 급식 만족도 조사학생들이 원하는 메뉴와 불만 사항을 파악하여 급식 메뉴 개선에 활용
🗳️
선거 여론 조사대통령·국회의원 선거 전 유권자 지지율을 파악하는 전화·온라인 설문
🛒
소비자 만족도 조사제품 구매 후 별점·리뷰를 수집하여 상품 개발에 반영

아래 질문에 답해보세요. 응답이 누적될수록 실시간 집계 결과가 그래프로 나타납니다.

② 1~2시간0명

💡 생각해보기: 이 데이터로 무엇을 알 수 있나요? 응답자 수가 더 많아지면 결과가 어떻게 달라질까요?
⚠️ 좋은 설문 문항 vs 나쁜 설문 문항
❌ 나쁜 예시
• "우리나라 교육이 너무 힘들지 않나요?"
→ 응답자를 특정 방향으로 유도 (편향 질문)
• "수학과 영어 공부를 좋아하나요?"
→ 두 과목을 한 문항에 묶음 (이중 질문)
• "당신의 의견을 자세히 써주세요."
→ 응답 기준이 모호 (불명확한 질문)
✅ 좋은 예시
• "오늘 급식에 만족하셨나요?" (5점 척도)
→ 중립적이고 측정 가능한 응답 설계
• "가장 좋아하는 과목은 무엇인가요?"
→ 하나의 주제, 명확한 선택지
• "하루 운동 시간은 몇 분인가요?"
→ 구체적이고 측정 가능한 응답
Method 02 · Observation

관찰

직접 보고, 듣고, 세면서 데이터를 기록하는 방법.
자연 현상, 사람의 행동, 사물의 변화 등을 체계적으로 기록합니다.

👁️
📖 핵심 개념
1
관찰 목적 설정 — 무엇을, 언제, 어디서 관찰할지 계획을 세운다.
2
기록 도구 준비 — 관찰 일지, 체크리스트, 카메라, 초시계 등을 준비한다.
3
체계적 기록 — 정해진 형식에 따라 빠짐없이 기록하여 누락을 방지한다.
4
반복·비교 — 여러 번 관찰하여 패턴을 찾고, 신뢰도를 높인다.
⚖️ 장점 & 단점
👍 PROS
실제로 일어난 일을 있는 그대로 기록 가능
응답자 편향이 없는 객관적 데이터
언어 장벽 없이 동·식물 등도 관찰 가능
행동·현상을 정량화할 수 있다
👎 CONS
관찰자의 주관이 개입될 수 있다
많은 시간·인력이 필요하다
관찰 대상이 의식하면 자연스럽지 않다
발생 빈도가 낮은 현상은 포착 어려움
🌍 실생활 활용 사례
🐦
조류 개체 수 조사특정 지역에 서식하는 새의 종류와 수를 직접 세어 생태 데이터 수집
🚗
교통량 조사특정 도로의 시간대별 차량 수를 세어 신호 체계 개선에 활용
🛒
매장 고객 동선 분석고객이 매장에서 어떤 경로로 이동하는지 관찰하여 진열 최적화

Method 03 · Sensor

센서 수집

전자 장치가 자동으로 환경 데이터를 측정·저장하는 방법.
온도·습도·위치·속도 등을 사람 없이도 24시간 수집합니다.

📡
📖 핵심 개념
1
센서 감지 — 온도·빛·압력 등 물리량을 전기 신호로 변환한다.
2
디지털 변환 (A/D) — 아날로그 신호를 컴퓨터가 읽을 수 있는 숫자로 바꾼다.
3
자동 저장 — 설정된 시간 간격으로 데이터를 서버나 로컬 메모리에 기록한다.
4
전송 & 분석 — 수집된 시계열 데이터를 인터넷으로 전송하여 분석한다 (IoT).
⚖️ 장점 & 단점
👍 PROS
24시간 자동 수집 — 사람 개입 불필요
초 단위로 정밀한 시계열 데이터 확보
인간이 직접 측정하기 어려운 환경도 가능
측정 오차가 일정하여 신뢰도 높음
👎 CONS
센서·인프라 설치 비용이 높다
고장 시 데이터 공백 발생
보정(캘리브레이션) 없으면 오차 누적
대용량 데이터 저장·처리 비용 필요
🌍 센서 종류와 활용 사례
🌡️
온도·습도 센서
기상 관측, 냉동창고 모니터링, 스마트팜 환경 제어
📍
GPS 위치 센서
내비게이션, 배달 추적, 야생동물 이동 경로 조사
💨
미세먼지 센서
에어코리아 대기질 측정망, 공기청정기 자동 제어
❤️
생체 센서
스마트워치 심박수·혈중 산소·수면 패턴 측정
⚡
전력 센서
스마트 미터기 전력 사용량 실시간 모니터링
📷
이미지 센서
교통 CCTV 차량 감지, 안면 인식, 자율주행 카메라

Method 04 · Open Public Data

공공데이터 포털

정부·공공기관이 이미 수집한 데이터를 누구나 무료로 내려받아 활용하는 방법.
data.go.kr에서 수만 개의 데이터셋을 검색할 수 있습니다.

🏛️
📖 핵심 개념
1
포털 검색 — data.go.kr 에서 필요한 데이터 키워드를 검색한다.
2
데이터 확인 — 파일(CSV/Excel) 또는 API 형태로 제공되는지 확인한다.
3
다운로드 or API 호출 — 파일을 직접 받거나 API로 실시간 데이터를 받아온다.
4
활용 — 스프레드시트·파이썬 등으로 분석하거나 앱에 연동한다.
⚖️ 장점 & 단점
👍 PROS
무료로 방대한 양의 데이터 이용 가능
국가기관이 수집하여 신뢰도 높음
오랜 기간의 과거 데이터도 확보 가능
직접 수집 없이 즉시 분석 시작 가능
👎 CONS
원하는 항목이 없을 수 있다
데이터 갱신 주기가 느릴 수 있다
형식이 불규칙하여 전처리 필요
공개 범위 제한으로 세부 정보 부재
