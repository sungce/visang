1. 컴퓨팅 시스템의 이해와 구성

교과서 9~11쪽

컴퓨팅 시스템은 눈에 보이는 기계 장치인 하드웨어와 이를 동작시키는 소프트웨어가 유기적으로 결합하여 문제를 해결하는 장치입니다.

하드웨어 (Hardware)

  • 입력 장치

    외부 데이터를 입력받음 (키보드, 마우스, 카메라)

  • 처리 장치

    명령어 해석 및 데이터 처리 (CPU, GPU, SoC)

  • 기억 장치

    명령어와 데이터 저장 (RAM, 하드 디스크, SSD)

  • 출력 장치

    처리 결과를 제공 (모니터, 프린터, 스피커)

  • 통신 장치

    데이터를 주고받음 (무선 공유기, 블루투스)

소프트웨어 (Software)

  • 시스템 소프트웨어 (운영 체제)

    하드웨어를 관리/운영하고 응용 소프트웨어가 동작할 수 있도록 기본적인 기능을 제공합니다.
    예: Windows, macOS, 안드로이드, iOS

2 문제 분석: 가속도 센서의 이해

교과서 19~20쪽

가속도 센서 (Accelerometer)

가속도 센서는 물체의 가속도나 충격의 세기를 측정하는 센서입니다. X축(좌우), Y축(앞뒤), Z축(위아래)의 미세한 움직임을 감지하여 자동차, 선박, 로봇 등에서 물체의 운동 상태와 흔들림을 파악하는 데 사용됩니다.

마이크로비트를 위아래로 흔들 때 발생하는 Z축의 변화를 감지하여 스쿼트 1회를 카운트할 수 있습니다!

3 알고리즘 설계

교과서 20쪽

큰 문제를 작은 문제로 나누기

작은 문제설명 (해결 방법)
1. 초기화변수를 생성하고 초깃값을 0으로 설정
2. 흔들림 감지하기몸의 흔들림(가속도 센서) 감지 시 변수 값 1 증가
3. 운동 횟수 안내하기A 버튼을 누르면 현재 '운동 횟수' 변수를 LED에 출력
4. 운동 횟수 재설정하기B 버튼을 누르면 '운동 횟수' 변수를 다시 0으로 재설정

알고리즘 순서도 구상

1
'운동 횟수' 변수를 생성하고 0으로 초기화한다.
2
흔들림이 감지되면 '운동 횟수' 변수를 1 증가시킨다.
3
A 버튼을 누르면 현재 '운동 횟수' 변수를 LED 스크린에 출력한다.
4
B 버튼을 누르면 '운동 횟수' 변수를 0으로 재설정한다.
