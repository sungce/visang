02

개념설명

⏱ 15분▾

문제 상황 & 분석

앉았다 일어서기 운동을 반복할 때 몇 번 했는지 자동으로 감지하여 알려주는 장치가 필요합니다.

  • 사용 센서: 가속도 센서 (흔들림 감지)
  • A 버튼: 현재 운동 횟수 LED 출력
  • B 버튼: 운동 횟수 초기화(0으로)
가속도 센서: 물체의 가속도·충격 세기 측정.
X·Y·Z 3축의 움직임을 미세하게 감지.

알고리즘 설계 (4단계)

➊ '운동 횟수' 변수를 생성하고 0으로 초기화한다.
➋ 흔들림이 감지되면 '운동 횟수' 변수를 1 증가시킨다.
➌ A 버튼을 누르면 현재 '운동 횟수'를 LED 스크린에 출력한다.
➍ B 버튼을 누르면 '운동 횟수' 변수를 0으로 재설정한다.

MakeCode 블록 예시

🟢 시작하면
운동횟수 ← 0
📳 흔들림 감지 시
운동횟수 ← 운동횟수 + 1
🅰️ A 버튼 누를 때
LED에 운동횟수 표시
🅱️ B 버튼 누를 때
운동횟수 ← 0
LED 화면 지우기
