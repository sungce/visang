UNIT Ⅰ — 2-02

🌙 수면 온도 알림이 장치 만들기

밤에 방 안의 온도를 자동으로 감지하여 LED에 표시하고, 수면에 적합하지 않은 온도일 때 알림을 주는 피지컬 컴퓨팅 시스템을 설계하고 프로그래밍합니다.

2차시 · 90분
교과서 24~27쪽
마이크로비트 / DHT11
01
학습 목표
2차시 · 이 소단원을 마치면 할 수 있어요
▶
[9정01-04] 2022 개정 성취기준
  • 수면과 온도의 관계를 이해하고, 쾌적한 수면 온도 범위(18~22℃)를 설명할 수 있다.
  • 온도 측정에 사용되는 센서(NTC 서미스터·LM35·DHT11·마이크로비트 내장)의 특징을 비교하고, 문제 해결에 적합한 센서를 선택할 수 있다.
  • 온도 조건에 따라 다른 출력을 하는 다중 조건 알고리즘(if~elif~else)을 설계하고 블록 코딩으로 구현할 수 있다.
  • 마이크로비트 LED·내장 온도 센서를 활용해 밤 방 안의 온도를 표시·알림하는 프로그램을 작성할 수 있다.

🥶저온 경고 (17℃ 미만)

추운 환경은 혈관 수축과 근육 긴장을 유발합니다. 마이크로비트 LED에 파란색 눈 결정(❄) 아이콘을 표시하고, 15℃ 이하에서는 부저로 경고합니다.

😴수면 적정 (18~22℃)

체온이 자연스럽게 내려가 깊은 수면 단계로 진입하기 좋습니다. LED에 녹색 하트(♥) 또는 온도 숫자를 표시합니다.

🥵고온 경고 (23℃ 초과)

더운 환경은 자주 깨게 하고 땀으로 탈수를 유발합니다. LED에 빨간색 태양(☀) 아이콘을 표시하고, 30℃ 이상에서는 부저로 즉시 경고합니다.

⚖️ 이번 단원 vs 이전 단원 코드 비교
📌 이전 단원: 운동 카운터 (단순 if)
# 조건 1가지만 처리 if accel > 700: count += 1 # if 하나로 끝
🆕 이번 단원: 온도 알림 (if~elif~else)
# 조건 3가지를 순서대로 처리 if temp < 18: # 첫째 조건 # 저온 처리 elif temp <= 22: # 둘째 조건 # 적정 처리 else: # 나머지 전부 # 고온 처리
🔑 elif의 핵심: if의 조건이 거짓일 때만 elif를 확인합니다. elif의 조건도 거짓이면 else가 실행됩니다. 조건이 많아도 단 하나의 블록만 실행됩니다.
