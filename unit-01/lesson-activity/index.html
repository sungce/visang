02

개념설명

⏱ 20분▾

라디오 통신(Radio)

무선 통신 방식 중 하나로, 같은 그룹(0~255)으로 설정한 여러 마이크로비트에 동시에 신호를 전달합니다.

  • 그룹 설정: 0~255 중 하나. 게임 참가자만 같은 번호
  • 전송: 숫자·문자를 라디오로 송신
  • 수신: 같은 그룹에서 보낸 숫자 수신
  • 동시 전송: 같은 그룹 전체에 신호 전달
⚠️ 게임에 참가할 두 명만 같은 그룹 번호를 설정해야 합니다!

알고리즘 설계 (4단계)

➊ 시작 → 라디오 그룹 1로 설정, '선택 결과'·'수신 결과' 변수 0으로 초기화
➋ A 버튼 → 1 전송 + '선택 결과'=1 저장 + '키득키득' 소리
   B 버튼 → 2 전송 + '선택 결과'=2 저장 + '키득키득' 소리
➌ 라디오 수신 → '수신 결과'에 수신 숫자 저장
➍ 두 변수 곱 > 0 이면 → 같으면 ♥, 다르면 ✕ LED 출력
