1. 컴퓨팅 시스템의 이해와 구성

컴퓨팅 시스템은 눈에 보이는 기계 장치인 하드웨어와 이를 동작시키는 소프트웨어가 유기적으로 결합하여 문제를 해결하는 장치입니다.

하드웨어 (Hardware)

  • 입력 장치

    외부 데이터를 입력받음 (키보드, 마우스, 카메라)

  • 처리 장치

    명령어 해석 및 데이터 연산 (CPU, GPU)

  • 기억 장치

    명령어와 데이터 저장 (RAM, SSD, HDD)

  • 출력 장치

    처리 결과를 사용자에게 제공 (모니터, 스피커)

  • 통신 장치

    데이터를 주고받음 (무선 공유기, 랜카드)

소프트웨어 (Software)

  • 시스템 소프트웨어 (운영 체제)

    하드웨어를 효율적으로 관리하고 응용 소프트웨어가 잘 동작할 수 있도록 돕는 가장 기본적인 프로그램입니다.

    Windows macOS Android
  • 응용 소프트웨어

    사용자가 특정한 목적을 달성하기 위해 사용하는 프로그램입니다.

2. 운영 체제와 컴퓨팅 시스템의 동작 원리

운영 체제(OS)는 입·출력 관리, 처리 장치 및 메모리 관리, 통신 장치 관리 기능을 수행합니다. 스마트폰의 동영상 재생 앱 실행 과정을 통해 전체 동작 원리를 살펴봅시다.

1. 입력

화면 터치로 앱 실행

운영 체제 (OS)

2. 처리

명령 해석 및 메모리 할당

3. 통신

서버에서 영상 다운로드

4. 출력

화면 재생 및 소리 출력
