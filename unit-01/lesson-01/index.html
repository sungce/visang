02

개념설명

⏱ 20분▾

① 컴퓨팅 시스템이란?

소프트웨어와 하드웨어가 유기적으로 결합하여 문제 해결에 활용되는 장치. 스마트폰·노트북·신호등 시스템·인공지능 스피커 등이 모두 컴퓨팅 시스템입니다.

② 하드웨어 5가지 장치

장치 역할 컴퓨터 예 스마트기기 예
입력 장치 외부 데이터 입력 키보드, 마우스, 웹캠, 마이크 터치스크린, 카메라, 터치 펜
처리 장치 명령어 해석·처리 CPU, GPU SoC(CPU+GPU 통합)
기억 장치 명령어·데이터 저장 HDD, SSD, RAM, USB RAM, SD카드
출력 장치 처리 결과 제공 모니터, 스피커, 프린터 디스플레이, 스피커, 진동모터
통신 장치 유무선 데이터 송수신 랜카드, 무선공유기 NFC, 블루투스, Wi-Fi

③ 소프트웨어 구분

시스템 소프트웨어
하드웨어를 관리·운영하여 응용 소프트웨어가 동작할 수 있도록 기본 기능 제공
대표: 운영 체제(OS)
• 컴퓨터: Windows, Linux, macOS
• 스마트기기: Android, iOS
응용 소프트웨어
특정 목적에 따라 사용자가 작업할 수 있도록 도움
예: 문서작성·사진편집·동영상편집·웹브라우저
• 스마트기기: 캘린더 앱, 음악 앱, 메시지 앱
