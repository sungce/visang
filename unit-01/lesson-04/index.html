02

개념설명

⏱ 15분▾
🌡️ 온도 센서
측정 범위: -5 ~ 50℃
주변 온도를 수치로 측정
계절별 실내 적정 온도:
봄·가을: 19~23℃ / 여름: 24~27℃ / 겨울: 18~21℃
☀️ 빛 감지 센서
측정 범위: 0(어두움) ~ 255(밝음)
주변 밝기로 밤·낮 구분
기준값 예: 128
128 미만 = 밤 / 128 이상 = 낮

알고리즘 설계 (3단계)

➊ A 버튼 → 현재 온도를 LED 스크린에 출력
➋ B 버튼 → 빛 감지 센서로 밤(🌙)/낮(☀️) 아이콘 표현
➌ 흔들림 감지 → 밤이면 온도에 따라 ↓(낮음)/🫀(적정)/↑(높음) 표시

MakeCode 블록 (핵심: 야간 온도 안내)

LED: ↑ (온도 높음)
